## 安装与首次打开

1. 从浏览器扩展商店安装（或从 Release 下载后手动安装）。
2. 安装完成后，首次会自动打开设置页；后续也可以通过以下方式进入：
   - 在扩展图标上右键 →「选项 / 设置」
   - 在 HamHome 面板/页面中点击设置入口

## 快速保存书签

你可以通过以下任意方式打开「保存面板」并收藏当前网页：

- 点击浏览器工具栏中的 HamHome 图标
- 在网页空白处右键 →「收藏到 HamHome」
- 使用快捷键（可在浏览器里自定义；默认建议为 `Ctrl+Shift+X` / `Command+Shift+X`）

在保存面板中，你可以：

- 编辑/确认 `标题`、`摘要`、`分类`、`标签`
- 点击「获取推荐」让 AI 生成摘要、推荐分类与标签（需要先在设置中配置 AI）
- 保存网页快照（若已开启「自动保存快照」，则收藏时会自动保存）

提示：如果当前页面已被收藏，保存面板会提示“已收藏”，你可以直接更新信息。

## 打开书签面板（网页内侧边栏）

HamHome 支持在任意网页内打开侧边栏面板，用于快速检索与打开书签：

- 快捷键切换面板（默认建议为 `Ctrl+Shift+L` / `Command+Shift+L`）
- 将鼠标移动到页面边缘，出现触发条后点击打开

你可以在设置中调整侧边栏显示在左侧或右侧。

## 书签搜索（关键词 / AI 对话）

在书签面板/管理页的搜索栏中：

- **关键词搜索**：适合精确匹配标题、URL、标签等
- **AI 对话式搜索**：切换到 AI 模式后，可用自然语言提问（例如“帮我找上周收藏的 React 相关文章”）

AI 对话模式下，回答会给出引用来源（可点击跳转到对应书签），并提供下一步建议。

## 分类与标签管理

### 分类

在「分类」页面中，你可以新建/编辑/删除分类，并支持多级分类结构。

如果你还没有分类体系，可以：

- 一键使用「预设分类」（通用型 / 专业创作者 / 技术向）
- 使用「AI 生成分类」：描述你的使用场景，让 AI 推荐一套分类结构（需配置 AI）

### 标签

在「标签」页面中，你可以查看并管理所有标签（包含统计与标签云），也可以在保存书签时随时添加新标签。

如果你希望 AI 推荐更稳定的标签风格，可在 AI 设置里配置「预设标签」，AI 会优先从预设标签中匹配并推荐。

## 网页快照（离线查看）

网页快照会将网页 HTML 内容保存在本地，用于：

- 原网页失效时仍可查看内容
- 离线阅读（取决于页面资源情况）

你可以在隐私/快照设置中开启或关闭「自动保存快照」。快照可能占用较多空间，建议定期在「存储管理」中查看占用并按需清理。

## 导入 / 导出与备份

在「导入/导出」页面中：

- **导出 JSON**：完整备份（用于迁移/恢复）
- **导出 HTML**：可直接打开浏览的静态网页格式
- **导入**：支持导入 HamHome 导出的 JSON，以及从 Chrome/Firefox/Edge 导出的书签 HTML

导入时常用选项：

- 「保留书签目录结构」：将浏览器文件夹结构映射为分类
- 「使用 AI 自动分类打标」：让 AI 在导入时为书签生成摘要/分类/标签（与保留目录结构互斥）
- 「获取页面内容进行分析」：会访问书签 URL 获取页面内容以提升 AI 准确性（更慢）

## 隐私与安全

- **本地优先**：书签数据与配置存储在浏览器本地（Chrome Storage + IndexedDB）
- **API Key 本地保存**：AI 的 API Key 仅保存在本地，直接与对应服务商通信
- **隐私域名**：你可以在隐私设置中添加域名（如 `example.com`），让这些域名的书签跳过 AI 分析

当你启用 AI 分析功能时，可能会向 AI 服务商发送以下信息用于生成摘要/标签等：

- 网页 URL
- 网页标题
- 网页内容摘要（用于生成描述和标签）

## 快捷键与自定义

HamHome 支持配置扩展快捷键（不同浏览器入口略有差异）：

- Chrome：`chrome://extensions/shortcuts`
- Edge：`edge://extensions/shortcuts`
- Firefox：在 `about:addons` 的扩展管理页中查找「管理扩展快捷键」

建议配置两类快捷键：

- 打开保存面板（收藏当前页）
- 切换网页内侧边栏面板

## 常见问题

### 1) 提示「AI 未配置」

你仍然可以手动保存书签；若需要 AI 的摘要/分类/标签/对话搜索，请到「设置 → AI 配置」填写服务商与 API Key 并测试连接。

### 2) AI 搜索效果一般 / 找不到结果

优先检查：

- 是否已开启并完成「语义搜索（Embedding）」索引构建（设置里可查看覆盖率与重建）
- 书签是否有足够的标题/摘要/内容（首次导入或早期收藏可能信息不完整）

### 3) 存储占用越来越大

快照会占用空间。可以在「隐私设置」里关闭自动保存快照，并在「存储管理」中导出备份后按需清理数据。
